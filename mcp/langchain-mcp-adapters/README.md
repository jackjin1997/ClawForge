# LangChain MCP Adapters
Integration documentation and reference for using MCP servers with LangChain.

## Usage
1. Install: `pip install langchain-mcp-adapters`
2. Configure MultiServerMCPClient to bridge stdio/http/sse MCP servers to LangChain Tools.

Refer to https://github.com/langchain-ai/langchain-mcp-adapters for implementation details.
