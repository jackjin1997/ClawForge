# Movie Oracle MCP

This directory contains resources for connecting OpenClaw to movie databases.

## Suggested MCP Servers

- **IMDb/Douban Search**: Use a search-engine-based MCP or a specific API wrapper to fetch movie details.
- **Recommendations**: System prompts designed to combine user preferences with data from multiple ratings sources.
